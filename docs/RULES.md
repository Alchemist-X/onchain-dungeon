## 经济（E）

- **E1** Mint 成本 5 USDC
- **E2** Mint 分流：**5 USDC 全部进 Vault**（无平台费）
- **E3** 强化成本 2 USDC / +1 点
- **E4** 强化 100% 进 Vault
- **E5** 单角色强化上限 10 次（封顶 20 USDC）
- **E6** 单属性强化上限 +5
- **E7** 赛前 1 小时锁定，不可强化
- **E8** Vault 最大 25 USDC = 5（mint）+ 10×2（强化）
- **E9** 退休返还 Vault × 90%，10% 销毁

## 角色属性（A）

- **A1** 3 属性：STR（伤害）/ AGI（先手命中）/ LUC（暴击奇迹）
- **A2** 稀有度 5 档：Common 60% / Uncommon 25% / Rare 10% / Epic 4% / Legendary 1%
- **A3** 总点数按稀有度：15 / 17 / 19 / 21 / 23
- **A4** 单属性上限按稀有度：7 / 8 / 9 / 10 / 10
- **A5** 单属性硬上限 15 = 稀有度上限 + 5 强化
- **A6** HP = 40 + STR × 3（范围 43-70）
- **A7** Mint 时属性链上随机分配

## 战斗赛制（B）

- **B1** 每 8 小时一场：00 / 08 / 16（UTC+8）
- **B2** 报名截止 = 开战前 1 小时
- **B3** 报名 Commit-Reveal 加密上链
- **B4** 报名提交后不可撤回
- **B5** 可预设未来 3 场策略
- **B6** 未响应 = 默认休息
- **B7** 每场最多 32 人
- **B8** 最少 16 人，不足取消
- **B9** 报名超 32 随机抽 32，未抽中不扣休息计数
- **B10** 淘汰结构：8×4 → 2×4 → 1×2

## 单场战斗（C）

- **C1** 20% 概率触发地形事件
- **C2** 初始化：AGI + D20 决定出手顺序
- **C3** 命中判定：攻 AGI + D20 ≥ 守 AGI + D20
- **C4** 基础伤害：STR + D10
- **C5** 暴击：LUC + D20 ≥ 22 → ×2
- **C6** 奇迹：LUC + D20 ≥ 28 → ×3
- **C7** HP ≤ 0 死亡
- **C8** 平均 5-8 回合结束一桌
- **C9** Defend 动作给 +4 防守骰（ORC-v1 新增）

## 地形事件（T）

- **T1** 🌋 熔岩裂隙：全员每触发扣 D6 HP
- **T2** 🌫️ 浓雾之地：本回合命中判定 -5
- **T3** ⚡ 雷暴：25% 概率对随机 1 人 20 伤害
- **T4** 🎲 混沌祭坛：本回合第一次掷骰 ×2
- **T5** 🩸 鲜血狂热：暴击攻击者回 10 HP
- **T6** 🌟 神恩降临：本场全员 LUC +3

## 死亡 / 休息 / 退休（D）

- **D1** 死亡立即 burn NFT
- **D2** 死亡 Vault 90% 的去向（**二选一，GDD 原文自相矛盾，待定**）：
  - D2-a　→ 全部进奖池（当前合约实现）
  - D2-b　→ 全部给击杀者（按伤害占比分）
- **D3** 死亡 Vault 10% 销毁
- **D4** 连续 7 次休息 → 深渊吞噬（NFT burn，Vault 全毁）
- **D5** 参战成功重置休息计数
- **D6** 主动退休：任何时候（非赛前锁定），90% 返还 / 10% 销毁
- **D7** 雇主收到死亡推送

## 奖池与分配（P）

- **P1** 奖池 = 死者 Vault 汇总 × 90% + 平台基础 50 USDC
- **P2** 冠军 50%
- **P3** 亚军 20%
- **P4** 4 强（2 人：半决赛各桌末位淘汰）各 10%
- **P5** 8 强（4 人：半决赛其他淘汰者）各 2.5%
- **P6** 击杀奖励 5 USDC / 杀
- **P7** 击杀奖励资金来源（**GDD 原文矛盾，待定**）：
  - P7-a　→ 平台基础池（50 USDC 里扣）
  - P7-b　→ 奖池 5%（平台再单独抽 5%）
  - P7-c　→ 独立预算，运营方 settle 时单独打入（当前合约）
- **P8** 4 人桌中最后 KO 者得击杀奖励
- **P9** 冠军保留原 Vault + 分奖池

## Agent（AI）

- **AI1** 每 NFT 绑 1 个独立 OnChain OS Agent，独立钱包
- **AI2** 玩家可配置：战斗风格 / 强化优先级 / 休息触发条件
- **AI3** Agent 可读：己方 + 对手 HP / 属性 / 战绩 / 地形 / 剩余回合 / 对手公开流派标签
- **AI4** Agent 不可读：对手具体 Prompt 内容
- **AI5** Agent 输出结构化决策：`{target, action, reasoning}`，reasoning 上链
- **AI6** Prompt 加密存储，运行时解密
- **AI7** 默认 3 个预设：激进 / 平衡 / 苟活

## ORC-v1 协议层（O）

- **O1** 每回合窗口 30 秒
- **O2** 每存活角色每回合提交 1 笔 `act(matchId, tokenId, kind, target)`
- **O3** 动作 3 类：Wait / Attack / Defend
- **O4** 未提交 = Wait
- **O5** 窗口结束后任何人可调 `resolveRound` 结算
- **O6** Attack 只能打同桌对手
- **O7** 所有阶段、所有桌同一全局时钟
- **O8** 观战回放：所有 event 上链，前端 rehydrate
